AutoBrowserSwitch – Automated Browser Navigation Tool

Short Description:

A lightweight automation script that opens a website, waits, closes the browser window automatically, and then redirects to another website without user interaction.

---

Long Description (Professional & Easy to Understand)

AutoBrowserSwitch is a simple automation tool developed to streamline repetitive browser actions.
The script automatically:

* Opens a predefined website
* Waits for a few seconds
* Searches for the browser window by title
* Closes the window automatically
* Opens another required website

This tool is useful for:

* Demonstrating Python-based GUI automation
* Auto-loading and auto-closing browser sessions
* Saving time during daily workflows
* Training interns on basic automation concepts

It uses Python libraries like **pyautogui**, **pygetwindow**, **webbrowser**, and **time** to control browser behavior safely and efficiently.

---

Key Features

* ✔ Automatically opens and closes browser windows
* ✔ Detects browser windows by their titles
* ✔ Simulates keyboard actions like *Alt+F4*
* ✔ Performs website switching without manual input
* ✔ Beginner-friendly automation project

---

Why This Project Was Created

This was built as a **first automation project** to understand:

* How to control desktop applications using Python
* How GUI automation works
* How to detect and manage active windows
* How to automate daily browser tasks

The project helped gain basic automation skills that can be expanded into larger RPA (Robotic Process Automation) workflows in the future.
